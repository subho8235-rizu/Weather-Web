# Weather-Web
#  Weather App

A real-time weather application that fetches data using the OpenWeather API.

## Features

* Search weather by city
* Displays temperature, condition, and icon
* Handles invalid input and errors
* Loading state for better UX

## Tech Stack

* HTML
* CSS
* JavaScript (Async/Await)
* OpenWeather API
